# 📩 Spam Detection System using Machine Learning

## 🔍 Overview

This project is a Spam Detection System that classifies messages as **Spam** or **Ham (Not Spam)** using Machine Learning techniques. It uses Natural Language Processing (NLP) and the Multinomial Naive Bayes algorithm.

---

## ⚙️ Technologies Used

* Python
* Pandas
* Scikit-learn
* CountVectorizer (NLP)

---

## 🚀 Features

* Text preprocessing and feature extraction
* Machine learning model training
* Spam/Ham classification
* User input prediction
* Model evaluation (Accuracy, Confusion Matrix)

---

## 🧠 Working

1. Dataset is loaded and cleaned
2. Text is converted into numerical format using CountVectorizer
3. Model is trained using Multinomial Naive Bayes
4. Predictions are made on test data
5. User can input custom messages for classification

---

## 📊 Output Example

* Accuracy Score
* Classification Report
* Confusion Matrix
* Spam/Ham Prediction

---

```bash
python naive bayes.py
```
